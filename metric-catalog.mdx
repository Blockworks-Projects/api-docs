## Overview

The Blockworks Research API provides access to a comprehensive catalog of crypto metrics across protocols, assets, and market data. This catalog helps you discover and understand the available metrics for your research and analysis.

<Note>
  The metric catalog is dynamically generated from our API. Use the `/v3/metrics` endpoint to get the most up-to-date list of available metrics.
</Note>

## Fetching Available Metrics

Use the metrics endpoint to get a real-time catalog of available metrics:

<CodeGroup>

```typescript TypeScript
const response = await fetch('https://rest.staging.blockworksresearch.com/v3/metrics', {
  headers: {
    'x-api-key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
console.log(`Found ${data.data.length} metrics`);

// Display metric information
data.data.forEach(metric => {
  console.log(`${metric.name}: ${metric.description}`);
  console.log(`Unit: ${metric.unit}`);
  console.log(`Projects: ${metric.projects?.join(', ')}`);
});
```

```python Python
import requests

headers = {
    'x-api-key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
}

response = requests.get(
    'https://rest.staging.blockworksresearch.com/v3/metrics',
    headers=headers
)

data = response.json()
print(f"Found {len(data['data'])} metrics")

# Display metric information
for metric in data['data']:
    print(f"{metric['name']}: {metric['description']}")
    print(f"Unit: {metric['unit']}")
    print(f"Projects: {', '.join(metric.get('projects', []))}")
```

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  https://rest.staging.blockworksresearch.com/v3/metrics
```

</CodeGroup>

## Querying Specific Metrics

Once you've identified a metric from the catalog, you can query its data:

### Example: Querying TVL for Uniswap

<CodeGroup>

```typescript TypeScript
const response = await fetch('https://rest.staging.blockworksresearch.com/v3/metrics/tvl', {
  headers: {
    'x-api-key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  params: {
    project: 'uniswap',
    limit: 30
  }
});

const data = await response.json();
console.log(`Retrieved ${data.data.length} TVL data points for Uniswap`);

// Process the data
data.data.forEach(point => {
  console.log(`${point.timestamp}: $${point.value.toLocaleString()}`);
});
```

```python Python
import requests

headers = {
    'x-api-key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
}

params = {
    'project': 'uniswap',
    'limit': 30
}

response = requests.get(
    'https://rest.staging.blockworksresearch.com/v3/metrics/tvl',
    headers=headers,
    params=params
)

data = response.json()
print(f"Retrieved {len(data['data'])} TVL data points for Uniswap")

# Process the data
for point in data['data']:
    print(f"{point['timestamp']}: ${point['value']:,.2f}")
```

</CodeGroup>

## Filtering and Searching

### Filter by Project

Get metrics available for a specific project:

```typescript
const response = await fetch('https://rest.staging.blockworksresearch.com/v3/metrics', {
  headers: {
    'x-api-key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  params: {
    project: 'uniswap'
  }
});
```

### Search by Identifier

Find metrics by their identifier pattern:

```python
import requests

# Get all metrics
response = requests.get('https://rest.staging.blockworksresearch.com/v3/metrics', 
                       headers={'x-api-key': 'YOUR_API_KEY'})
metrics = response.json()['data']

# Filter metrics containing 'volume'
volume_metrics = [m for m in metrics if 'volume' in m['identifier']]

for metric in volume_metrics:
    print(f"{metric['name']}: {metric['identifier']}")
```

## Metric Data Structure

Each metric in the catalog has the following structure:

```json
{
  "identifier": "tvl",
  "name": "Total Value Locked",
  "description": "Total value locked across DeFi protocols",
  "unit": "USD",
  "projects": ["uniswap", "aave", "compound", "makerdao"],
  "category": "defi",
  "frequency": "daily",
  "data_availability": {
    "start_date": "2020-01-01",
    "end_date": "2024-01-01"
  }
}
```

## Common Use Cases

### Portfolio Analytics

Track key metrics for your portfolio holdings:

```typescript
const portfolioMetrics = ['tvl', 'volume_24h', 'daily_active_users'];
const projects = ['uniswap', 'aave', 'compound'];

// Fetch metrics for all projects
const metricData = await Promise.all(
  portfolioMetrics.map(metric =>
    projects.map(project =>
      fetch(`/v3/metrics/${metric}?project=${project}`)
    )
  )
);
```

### Research Dashboard

Build a comprehensive research dashboard:

```python
# Key metrics to track
research_metrics = [
    'market_cap',
    'tvl', 
    'daily_active_users',
    'transaction_count',
    'fees_24h'
]

# Fetch data for analysis
dashboard_data = {}
for metric in research_metrics:
    response = requests.get(f'/v3/metrics/{metric}', 
                           headers={'x-api-key': 'YOUR_API_KEY'})
    dashboard_data[metric] = response.json()['data']
```
