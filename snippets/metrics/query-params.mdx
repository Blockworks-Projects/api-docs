### Query Parameters

| Parameter | Required | Description | Example |
|---|:---:|---|---|
| `project` | **Yes** | Projects to include in the response (comma-separated) | `Ethereum,Bitcoin` |
| `start_date` | No | Start date (YYYY-MM-DD) | `2025-06-06` |
| `end_date` | No | End date (YYYY-MM-DD) | `2025-07-24` |
