REST API providing crypto asset data and blockchain metrics. Access fee structures, issuance data, revenue metrics, and comprehensive asset information for your applications and research.


## Sample Request

```bash
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworksresearch.com/v3/assets"
```
