## Quick Start

### 1. Authentication

All API requests require authentication using your API key. Include it in the `x-api-key` header:

```bash
curl -H "x-api-key: YOUR_API_KEY" \
  https://rest.staging.blockworksresearch.com/v3/assets
```

### 2. Your First Request

Let's fetch the list of available assets:

<CodeGroup>

```typescript TypeScript
const response = await fetch('https://rest.staging.blockworksresearch.com/v3/assets', {
  headers: {
    'x-api-key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

headers = {
    'x-api-key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
}

response = requests.get(
    'https://rest.staging.blockworksresearch.com/v3/assets',
    headers=headers
)

data = response.json()
print(data)
```

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  https://rest.staging.blockworksresearch.com/v3/assets
```

</CodeGroup>

### 3. Response Format

All API responses follow a consistent structure:

```json
{
  "data": [...],
  "pagination": {
    "page": 1,
    "limit": 10,
    "total": 1000,
    "pages": 100
  }
}
```
