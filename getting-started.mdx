## Quick Start


### 1. Get an API Key
Log in to [Blockworks Research](https://www.blockworksresearch.com/) and go to the [API Keys](https://www.blockworksresearch.com/api-keys) page.

Click on "Create API Key" and give your key a name.
Copy this key {API_BASE} and save it somewhere safe.
We'll be using it in the next steps!



### 2. Make a Request

Let's fetch the list of available assets:

<CodeGroup>

```typescript TypeScript
const data = await fetch(
  'https://api.blockworksresearch.com/v3/assets',
  { headers: { 'x-api-key': 'YOUR_API_KEY' } }
)

const data = await response.json()
```

```python Python
import requests

response = requests.get(
    'https://api.blockworksresearch.com/v3/assets',
    headers = { 'x-api-key': 'YOUR_API_KEY' }
)

data = response.json()
```

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  https://api.blockworksresearch.com/v3/assets
```

</CodeGroup>

### 3. Example Response

a `total` property indicating the total number of items,
and a `page` property indicating the current page.
