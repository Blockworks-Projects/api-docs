title: "Treasury Holdings"
description: "Number of tokens held by the company. See crypto-asset for asset indicator"
openapi: 'GET /v1/metrics/treasury-token-holdings'
---

## Overview

- **Unit:** native units
- **Interval:** daily
- **Source:** Blockworks

## Example Request

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworks.com/v1/metrics/treasury-token-holdings?project=SBET"
```

```typescript TypeScript
const res = await fetch(
  'https://api.blockworks.com/v1/metrics/treasury-token-holdings?project=SBET',
  { headers: { 'x-api-key': 'YOUR_API_KEY' } }
)
const data = await res.json()
```

```python Python
import requests

r = requests.get(
  'https://api.blockworks.com/v1/metrics/treasury-token-holdings',
  headers={'x-api-key': 'YOUR_API_KEY'},
  params={'project': 'SBET'}
)
data = r.json()
```
