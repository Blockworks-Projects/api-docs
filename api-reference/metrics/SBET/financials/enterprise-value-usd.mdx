---
title: "Enterprise Value USD"
description: "Actual enterprise value for the equity. Calculated as market cap - liabilities + assets"
openapi: 'GET /v1/metrics/enterprise-value-usd'
---

## Overview

- **Unit:** USD
- **Interval:** daily
- **Source:** Blockworks

## Example Request

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworks.com/v1/metrics/enterprise-value-usd?project=SBET"
```

```typescript TypeScript
const res = await fetch(
  'https://api.blockworks.com/v1/metrics/enterprise-value-usd?project=SBET',
  { headers: { 'x-api-key': 'YOUR_API_KEY' } }
)
const data = await res.json()
```

```python Python
import requests

r = requests.get(
  'https://api.blockworks.com/v1/metrics/enterprise-value-usd',
  headers={'x-api-key': 'YOUR_API_KEY'},
  params={'project': 'SBET'}
)
data = r.json()
```
