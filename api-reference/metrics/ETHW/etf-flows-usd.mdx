## Overview

- **Unit:** USD
- **Interval:** daily
- **Source:** Farside UK

## Example Request

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworks.com/v1/metrics/etf-flows-usd?project=ETHW"
```

```typescript TypeScript
const res = await fetch(
  'https://api.blockworks.com/v1/metrics/etf-flows-usd?project=ETHW',
  { headers: { 'x-api-key': 'YOUR_API_KEY' } }
)
const data = await res.json()
```

```python Python
import requests

r = requests.get(
  'https://api.blockworks.com/v1/metrics/etf-flows-usd',
  headers={'x-api-key': 'YOUR_API_KEY'},
  params={'project': 'ETHW'}
)
data = r.json()
```

</CodeGroup>
