## Basic Example

<CodeGroup>

```typescript TypeScript
const response = await fetch(
  'https://rest.staging.blockworksresearch.com/v3/metrics/txns?project=Ethereum', {
  headers: { 'x-api-key': 'YOUR_API_KEY' }
})

const data = await response.json()
```

```python Python
import requests

response = requests.get(
    'https://rest.staging.blockworksresearch.com/v3/metrics/txns',
    headers={'x-api-key': 'YOUR_API_KEY'},
    params={'project': 'Ethereum'}
)

data = response.json()
```

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://rest.staging.blockworksresearch.com/v3/metrics/txns?project=Ethereum"
```

</CodeGroup>


### Response Structure

Returns a specific metric for a project:

```json
{
  "Ethereum": {
    "date": "2025-07-23",
    "value": 294405
  }
}
```

## Configuration

### Path Parameters

| Parameter | Type | Description | Required |
|-----------|------|-------------|----------|
| `identifier` | `string` | Metric identifier | Yes |

### Query Parameters

| Parameter | Type | Description | Default |
|-----------|------|-------------|---------|
| `project` | string | Project identifier | Required |
| `start_date` | string | Start date (YYYY-MM-DD) | 30 days ago |
| `end_date` | string | End date (YYYY-MM-DD) | Today |
| `aggregation` | string | Data aggregation (daily, weekly, monthly) | daily |
| `format` | string | Response format (json, csv) | json |
