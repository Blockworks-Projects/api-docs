## Overview

- **Denomination:** %
- **Type:** Decimal
- **Interval:** Daily
- **Source:** Blockworks

## Example Request

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworks.com/v1/metrics/etf-bitcoin-aum-share?project=harvest-global-investments-limited"
```

```typescript TypeScript
const res = await fetch(
  'https://api.blockworks.com/v1/metrics/etf-bitcoin-aum-share?project=harvest-global-investments-limited',
  { headers: { 'x-api-key': 'YOUR_API_KEY' } }
)
const data = await res.json()
```

```python Python
import requests

r = requests.get(
  'https://api.blockworks.com/v1/metrics/etf-bitcoin-aum-share',
  headers={'x-api-key': 'YOUR_API_KEY'},
  params={'project': 'harvest-global-investments-limited'}
)
data = r.json()
```
