---
title: "mNAV"
description: "Market NAV value for that company. Calculated as enterprise value / NAV"
openapi: 'GET /v1/metrics/treasury-mnav'
---

## Overview

- **Unit:** native units
- **Interval:** daily
- **Source:** Blockworks

## Example Request

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworks.com/v1/metrics/treasury-mnav?project=ETHM"
```

```typescript TypeScript
const res = await fetch(
  'https://api.blockworks.com/v1/metrics/treasury-mnav?project=ETHM',
  { headers: { 'x-api-key': 'YOUR_API_KEY' } }
)
const data = await res.json()
```

```python Python
import requests

r = requests.get(
  'https://api.blockworks.com/v1/metrics/treasury-mnav',
  headers={'x-api-key': 'YOUR_API_KEY'},
  params={'project': 'ETHM'}
)
data = r.json()
```
