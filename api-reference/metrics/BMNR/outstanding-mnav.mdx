## Overview

- **Denomination:** mNAV
- **Type:** Decimal
- **Interval:** Daily
- **Source:** Blockworks

## Example Request

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworks.com/v1/metrics/outstanding-mnav?project=BMNR"
```

```typescript TypeScript
const res = await fetch(
  'https://api.blockworks.com/v1/metrics/outstanding-mnav?project=BMNR',
  { headers: { 'x-api-key': 'YOUR_API_KEY' } }
)
const data = await res.json()
```

```python Python
import requests

r = requests.get(
  'https://api.blockworks.com/v1/metrics/outstanding-mnav',
  headers={'x-api-key': 'YOUR_API_KEY'},
  params={'project': 'BMNR'}
)
data = r.json()
```
