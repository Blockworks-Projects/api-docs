---
title: "M NAV"
description: "Market NAV value for that company. Calculated as enterprise value / NAV"
openapi: 'GET /v1/metrics/treasury-mnav'
---

## Overview

- **Unit:** native units
- **Interval:** daily
- **Source:** Blockworks

## Example Response

```json
{
  "bmnr": [
    {
      "date": "2025-08-18",
      "value": 1.448112843453425

    {
      "date": "2025-08-14",
      "value": 1.961636086989526
    }
  ]
}
```

## Notes
- daily aggregates unless otherwise noted.
- Data is updated daily and may be revised after late-arriving data.
