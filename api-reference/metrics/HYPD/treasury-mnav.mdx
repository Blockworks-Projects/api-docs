## Overview

- **Unit:** native units
- **Interval:** daily
- **Source:** Blockworks

## Example Request

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworks.com/v1/metrics/treasury-mnav?project=HYPD"
```

```typescript TypeScript
const res = await fetch(
  'https://api.blockworks.com/v1/metrics/treasury-mnav?project=HYPD',
  { headers: { 'x-api-key': 'YOUR_API_KEY' } }
)
const data = await res.json()
```

```python Python
import requests

r = requests.get(
  'https://api.blockworks.com/v1/metrics/treasury-mnav',
  headers={'x-api-key': 'YOUR_API_KEY'},
  params={'project': 'HYPD'}
)
data = r.json()
```

</CodeGroup>
