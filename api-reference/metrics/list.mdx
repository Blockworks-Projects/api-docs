### Response Structure

Returns a list of metrics:

```json
{
  "data": [
    {
      "name": "Active Addresses",
      "description": "Not a measure of user growth",
      "identifier": "active-addresses",
      "project": "Avax",
      "source": "Blockworks",
      "data_type": "timeseries_int",
      "parameters": { /* available query parameters */ }
    }
  ],
  "total": 12,
  "page": 1
}
```

## Configuration

### Query Parameters

| Parameter | Type | Description | Default |
|-----------|------|-------------|---------|
| `page` | integer | Page number (1-indexed) | 1 |
| `limit` | integer | Items per page (max 100) | 10 |
| `project` | string | Filter by project identifier | - |
| `category` | string | Filter by metric category | - |
| `search` | string | Search metrics by name or description | - |

## Examples

### Filter by Project

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworksresearch.com/v3/metrics?project=uniswap"
```

```typescript TypeScript
const data = await fetch('https://api.blockworksresearch.com/v3/metrics?project=uniswap', {
  headers: {
    'x-api-key': 'YOUR_API_KEY',
  }
})
.then(r => r.json());
console.log(`Found ${data.data.length} metrics for Uniswap`);

data.data.forEach(metric => {
  console.log(`${metric.identifier}: ${metric.name}`);
});
```

```python Python
headers = {'x-api-key': 'YOUR_API_KEY'}
params = {'project': 'uniswap'}

response = requests.get(
    'https://api.blockworksresearch.com/v3/metrics',
    headers=headers,
    params=params
)

data = response.json()
print(f"Found {len(data['data'])} metrics for Uniswap")

for metric in data['data']:
    print(f"{metric['identifier']}: {metric['name']}")
```

</CodeGroup>

### Search Metrics

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworksresearch.com/v3/metrics?search=volume"
```

```typescript TypeScript
const data = await fetch('https://api.blockworksresearch.com/v3/metrics?search=volume', {
  headers: {
    'x-api-key': 'YOUR_API_KEY',
  }
})
.then(r => r.json());
console.log(`Found ${data.data.length} volume-related metrics`);

data.data.forEach(metric => {
  console.log(`${metric.identifier}: ${metric.name}`);
  console.log(`  Unit: ${metric.unit}`);
  console.log(`  Projects: ${metric.projects.join(', ')}`);
});
```

```python Python
headers = {'x-api-key': 'YOUR_API_KEY'}
params = {'search': 'volume'}

response = requests.get(
    'https://api.blockworksresearch.com/v3/metrics',
    headers=headers,
    params=params
)

data = response.json()
print(f"Found {len(data['data'])} volume-related metrics")

for metric in data['data']:
    print(f"{metric['identifier']}: {metric['name']}")
    print(f"  Unit: {metric['unit']}")
    print(f"  Projects: {', '.join(metric['projects'])}")
```
