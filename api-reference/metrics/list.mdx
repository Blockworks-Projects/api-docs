## Basic Example

<CodeGroup>

```typescript TypeScript
const response = await fetch(
  'https://api.blockworksresearch.com/v3/metrics', {
  headers: { 'x-api-key': 'YOUR_API_KEY' }
})

const data = await response.json()
```

```python Python
import requests

response = requests.get(
    'https://api.blockworksresearch.com/v3/metrics',
    headers = { 'x-api-key': 'YOUR_API_KEY' }
)

data = response.json()
```

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworksresearch.com/v3/metrics"
```


</CodeGroup>

### Response Structure

Returns a list of metrics:

```json
{
  "data": [
    {
      "name": "Active Addresses",
      "description": "Not a measure of user growth",
      "identifier": "active-addresses",
      "project": "Avax",
      "source": "Blockworks",
      "data_type": "timeseries_int",
      "parameters": { /* available query parameters */ }
    }
  ],
  "total": 12,
  "page": 1
}
```

## Configuration

### Query Parameters

| Parameter | Type | Description | Default |
|-----------|------|-------------|---------|
| `page` | integer | Page number (1-indexed) | 1 |
| `limit` | integer | Items per page (max 100) | 10 |
| `project` | string | Filter by project identifier | - |
| `category` | string | Filter by metric category | - |
| `search` | string | Search metrics by name or description | - |

## Examples

### Filter by Project

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworksresearch.com/v3/metrics?project=uniswap"
```

```typescript TypeScript
const response = await fetch(
  'https://api.blockworksresearch.com/v3/metrics?project=uniswap', {
  headers: { 'x-api-key': 'YOUR_API_KEY' }
})

const data = await response.json()
```

```python Python
response = requests.get(
    'https://api.blockworksresearch.com/v3/metrics',
    headers = { 'x-api-key': 'YOUR_API_KEY' },
    params = { 'project': 'uniswap' }
)

data = response.json()
```

</CodeGroup>
