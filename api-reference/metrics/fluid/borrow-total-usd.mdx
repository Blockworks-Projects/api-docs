## Overview

- **Denomination:** USD
- **Type:** USD Value
- **Interval:** Daily
- **Source:** Blockworks

## Example Request

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworks.com/v1/metrics/borrow-total-usd?project=fluid"
```

```typescript TypeScript
const res = await fetch(
  'https://api.blockworks.com/v1/metrics/borrow-total-usd?project=fluid',
  { headers: { 'x-api-key': 'YOUR_API_KEY' } }
)
const data = await res.json()
```

```python Python
import requests

r = requests.get(
  'https://api.blockworks.com/v1/metrics/borrow-total-usd',
  headers={'x-api-key': 'YOUR_API_KEY'},
  params={'project': 'fluid'}
)
data = r.json()
```
