## Example Request

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworks.com/v1/metrics/treasury-crypto-asset?project=CYFRD"
```

```typescript TypeScript
const res = await fetch(
  'https://api.blockworks.com/v1/metrics/treasury-crypto-asset?project=CYFRD',
  { headers: { 'x-api-key': 'YOUR_API_KEY' } }
)
const data = await res.json()
```

```python Python
import requests

r = requests.get(
  'https://api.blockworks.com/v1/metrics/treasury-crypto-asset',
  headers={'x-api-key': 'YOUR_API_KEY'},
  params={'project': 'CYFRD'}
)
data = r.json()
```
