## Overview

- **Denomination:** Count
- **Type:** Integer
- **Interval:** Daily
- **Source:** Blockworks

## Example Request

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworks.com/v1/metrics/launchpad-tokens-graduated?project=boopfun"
```

```typescript TypeScript
const res = await fetch(
  'https://api.blockworks.com/v1/metrics/launchpad-tokens-graduated?project=boopfun',
  { headers: { 'x-api-key': 'YOUR_API_KEY' } }
)

const data = await res.json()
```

```python Python
import requests

r = requests.get(
  'https://api.blockworks.com/v1/metrics/launchpad-tokens-graduated',
  headers={'x-api-key': 'YOUR_API_KEY'},
  params={'project': 'boopfun'}
)

data = r.json()
```
