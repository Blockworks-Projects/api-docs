## Overview

- **Denomination:** TIA
- **Type:** Decimal
- **Interval:** Daily
- **Source:** Blockworks

## Example Request

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworks.com/v1/metrics/token-supply-staked-native?project=celestia"
```

```typescript TypeScript
const res = await fetch(
  'https://api.blockworks.com/v1/metrics/token-supply-staked-native?project=celestia',
  { headers: { 'x-api-key': 'YOUR_API_KEY' } }
)
const data = await res.json()
```

```python Python
import requests

r = requests.get(
  'https://api.blockworks.com/v1/metrics/token-supply-staked-native',
  headers={'x-api-key': 'YOUR_API_KEY'},
  params={'project': 'celestia'}
)
data = r.json()
```
