---
title: "Native Token Price"
description: "The average token price during the specified time period for the native asset of that chain."
openapi: 'GET /v1/metrics/native-token-price'
---

## Overview

- **Unit:** USD
- **Aggregation:** Sum
- **Source:** Blockworks

## Example Response

```json
{
  "arbitrum": [
    {
      "date": "2025-08-10",
      "value": 0.46306694791666686
    },
    {
      "date": "2025-08-11",
      "value": 0.4566185833333327
    },
    {
      "date": "2025-08-12",
      "value": 0.4539177881944452
    },
    {
      "date": "2025-08-13",
      "value": 0.5118599791666653
    }
  ]
}
```

## Notes
- Sum aggregates unless otherwise noted.
- Data is updated daily and may be revised after late-arriving data.
