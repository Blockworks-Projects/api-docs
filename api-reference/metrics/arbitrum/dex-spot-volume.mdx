---
title: 'Dex Spot Volume'
description: 'Calculates daily spot volume from major DEXs. Only includes volumes from fee-paying liquidity pools'
openapi: 'GET /v1/metrics/dex-spot-volume'
---

## Overview

- **Unit:** USD
- **Aggregation:** daily
- **Source:** Blockworks



## Example Response

```json
{
  "arbitrum": [
    { "date": "2025-08-01", "value": 592.3947 },
    { "date": "2025-08-02", "value": 605.2811 },
    { "date": "2025-08-03", "value": 618.0204 },
    { "date": "2025-08-04", "value": 598.472 }
  ]
}
```

## Notes
- Daily aggregates unless otherwise noted.
- Data is updated daily and may be revised after late-arriving data.
