---
title: 'Fees Priority'
description: 'An additional transaction fee to incentivize validators to prioritize and execute transactions.'
openapi: 'GET /v1/metrics/fees-priority'
---

## Overview

- **Unit:** native units
- **Aggregation:** daily
- **Source:** Blockworks



## Example Response

```json
{
  "arbitrum": [
    { "date": "2025-08-01", "value": 592.3947 },
    { "date": "2025-08-02", "value": 605.2811 },
    { "date": "2025-08-03", "value": 618.0204 },
    { "date": "2025-08-04", "value": 598.472 }
  ]
}
```

## Notes
- Daily aggregates unless otherwise noted.
- Use USD conversion with [`fees-priority-usd`](/api-reference/metrics/arbitrum/fees-priority-usd) for fiat value.
- Data is updated daily and may be revised after late-arriving data.
