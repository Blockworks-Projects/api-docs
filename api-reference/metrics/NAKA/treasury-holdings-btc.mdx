## Overview

- **Denomination:** BTC
- **Type:** Decimal
- **Interval:** Daily
- **Source:** Blockworks

## Example Request

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworks.com/v1/metrics/treasury-holdings-btc?project=NAKA"
```

```typescript TypeScript
const res = await fetch(
  'https://api.blockworks.com/v1/metrics/treasury-holdings-btc?project=NAKA',
  { headers: { 'x-api-key': 'YOUR_API_KEY' } }
)
const data = await res.json()
```

```python Python
import requests

r = requests.get(
  'https://api.blockworks.com/v1/metrics/treasury-holdings-btc',
  headers={'x-api-key': 'YOUR_API_KEY'},
  params={'project': 'NAKA'}
)
data = r.json()
```

</CodeGroup>
