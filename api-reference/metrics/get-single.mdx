### Response Structure

The API returns metric data with the following structure:

```json
{
  "data": [
    {
      "date": "2024-01-15",
      "value": 150000,
      "project": "uniswap"
    },
    {
      "date": "2024-01-16", 
      "value": 152000,
      "project": "uniswap"
    }
  ],
  "metadata": {
    "metric": {
      "name": "Active Addresses",
      "description": "Number of unique addresses that interacted with the protocol",
      "identifier": "active-addresses",
      "data_type": "timeseries_int",
      "unit": "addresses"
    },
    "project": "uniswap",
    "start_date": "2024-01-15",
    "end_date": "2024-01-16",
    "total_points": 2
  }
}
```

## Making a Request

<CodeGroup>

```typescript TypeScript
const response = await fetch(
  'https://rest.staging.blockworksresearch.com/v3/metrics/active-addresses?project=uniswap',
  { headers: { 'x-api-key': 'YOUR_API_KEY' } }
)

const data = await response.json()
```

```python Python
import requests

response = requests.get(
    'https://rest.staging.blockworksresearch.com/v3/metrics/active-addresses',
    headers={'x-api-key': 'YOUR_API_KEY'},
    params={'project': 'uniswap'}
)

data = response.json()
```

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://rest.staging.blockworksresearch.com/v3/metrics/active-addresses?project=uniswap"
```

</CodeGroup>

### Path Parameters

| Parameter | Type | Description | Required |
|-----------|------|-------------|----------|
| `identifier` | `string` | Metric identifier | Yes |

### Query Parameters

| Parameter | Type | Description | Default |
|-----------|------|-------------|---------|
| `project` | string | Project identifier | Required |
| `start_date` | string | Start date (YYYY-MM-DD) | 30 days ago |
| `end_date` | string | End date (YYYY-MM-DD) | Today |
| `aggregation` | string | Data aggregation (daily, weekly, monthly) | daily |
| `format` | string | Response format (json, csv) | json |

<CodeGroup>


### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `data` | array | Array of metric data points |
| `data[].date` | string | Date in YYYY-MM-DD format |
| `data[].value` | number | Metric value for the date |
| `data[].project` | string | Project identifier |
| `metadata` | object | Additional information about the metric |
| `metadata.metric` | object | Metric definition and properties |
| `metadata.project` | string | Project identifier |
| `metadata.start_date` | string | Query start date |
| `metadata.end_date` | string | Query end date |
| `metadata.total_points` | number | Total number of data points returned |

### Common Use Cases

#### Building Time Series Charts

```typescript
async function buildChart(metric: string, project: string) {
  const response = await fetch(
    `https://rest.staging.blockworksresearch.com/v3/metrics/${metric}?project=${project}`,
    { headers: { 'x-api-key': 'YOUR_API_KEY' } }
  )
  
  const data = await response.json()
  
  // Format for chart library (e.g., Chart.js)
  const chartData = {
    labels: data.data.map(point => point.date),
    datasets: [{
      label: data.metadata.metric.name,
      data: data.data.map(point => point.value),
      borderColor: 'rgb(75, 192, 192)',
      tension: 0.1
    }]
  }
  
  return chartData
}
```

#### Comparing Multiple Projects

```python
def compare_projects(metric_id, projects, start_date, end_date):
    """Compare a metric across multiple projects"""
    comparison_data = {}
    
    for project in projects:
        response = requests.get(
            f'https://rest.staging.blockworksresearch.com/v3/metrics/{metric_id}',
            headers={'x-api-key': 'YOUR_API_KEY'},
            params={
                'project': project,
                'start_date': start_date,
                'end_date': end_date
            }
        )
        
        if response.status_code == 200:
            data = response.json()
            comparison_data[project] = data['data']
    
    return comparison_data

# Usage
projects = ['ethereum', 'bitcoin', 'solana']
tvl_comparison = compare_projects('tvl', projects, '2024-01-01', '2024-01-31')
```
