## Overview

- **Denomination:** PLASMA
- **Type:** Decimal
- **Interval:** Daily
- **Source:** Blockworks

## Example Request

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworks.com/v1/metrics/rev-native?project=plasma"
```

```typescript TypeScript
const res = await fetch(
  'https://api.blockworks.com/v1/metrics/rev-native?project=plasma',
  { headers: { 'x-api-key': 'YOUR_API_KEY' } }
)
const data = await res.json()
```

```python Python
import requests

r = requests.get(
  'https://api.blockworks.com/v1/metrics/rev-native',
  headers={'x-api-key': 'YOUR_API_KEY'},
  params={'project': 'plasma'}
)
data = r.json()
```
