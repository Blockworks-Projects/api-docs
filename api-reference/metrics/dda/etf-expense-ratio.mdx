## Overview

- **Denomination:** %
- **Type:** Decimal
- **Interval:** Daily
- **Source:** Blockworks

## Example Request

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworks.com/v1/metrics/etf-expense-ratio?project=dda"
```

```typescript TypeScript
const res = await fetch(
  'https://api.blockworks.com/v1/metrics/etf-expense-ratio?project=dda',
  { headers: { 'x-api-key': 'YOUR_API_KEY' } }
)
const data = await res.json()
```

```python Python
import requests

r = requests.get(
  'https://api.blockworks.com/v1/metrics/etf-expense-ratio',
  headers={'x-api-key': 'YOUR_API_KEY'},
  params={'project': 'dda'}
)
data = r.json()
```
