---
title: 'Burn USD'
description: 'The total amount of burned transaction fees in USD.'
openapi: 'GET /v1/metrics/burn-usd'
---

## Overview

- **Unit:** USD
- **Aggregation:** daily
- **Source:** Blockworks



## Example Response

```json
{
  "solana": [
    { "date": "2025-08-01", "value": 592.3947 },
    { "date": "2025-08-02", "value": 605.2811 },
    { "date": "2025-08-03", "value": 618.0204 },
    { "date": "2025-08-04", "value": 598.472 }
  ]
}
```

## Notes
- Values are reported in **USD** where applicable.
- Native counterpart: [`burn`](/api-reference/metrics/solana/burn)
- Data is updated daily and may be revised after late-arriving data.
