---
title: 'Issuance'
description: 'Newly issued tokens that are distributed to participants (such as validators, miners, or stakers) as compensation for participating in consensus.'
openapi: 'GET /v1/metrics/issuance'
---

## Overview

- **Unit:** native units
- **Aggregation:** daily
- **Source:** Blockworks



## Example Response

```json
{
  "solana": [
    { "date": "2025-08-01", "value": 592.3947 },
    { "date": "2025-08-02", "value": 605.2811 },
    { "date": "2025-08-03", "value": 618.0204 },
    { "date": "2025-08-04", "value": 598.472 }
  ]
}
```

## Notes
- Daily aggregates unless otherwise noted.
- Use USD conversion with [`issuance-usd`](/api-reference/metrics/solana/issuance-usd) for fiat value.
- Data is updated daily and may be revised after late-arriving data.
