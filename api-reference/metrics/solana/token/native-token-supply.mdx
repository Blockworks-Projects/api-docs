---
title: "Native Token Supply"
description: "The total token supply for the respective native asset of that chain. Inclusive of all locked tokens."
openapi: 'GET /v1/metrics/native-token-supply'
---

## Overview

- **Unit:** native units
- **Interval:** daily
- **Source:** Blockworks

## Notes
- daily aggregates unless otherwise noted.
- Data is updated daily and may be revised after late-arriving data.
