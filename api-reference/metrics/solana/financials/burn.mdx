## Overview

- **Unit:** Native units
- **Aggregation:** Sum
- **Source:** Blockworks

## Notes
- Sum aggregates unless otherwise noted.
- Data is updated daily and may be revised after late-arriving data.


<Note>For fiat values, see **[Burn USD](./burn-usd)**.</Note>
