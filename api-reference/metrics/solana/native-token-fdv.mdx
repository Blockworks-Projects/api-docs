## Overview

- **Unit:** USD
- **Interval:** daily
- **Source:** Blockworks

## Example Request

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworks.com/v1/metrics/native-token-fdv?project=solana"
```

```typescript TypeScript
const res = await fetch(
  'https://api.blockworks.com/v1/metrics/native-token-fdv?project=solana',
  { headers: { 'x-api-key': 'YOUR_API_KEY' } }
)
const data = await res.json()
```

```python Python
import requests

r = requests.get(
  'https://api.blockworks.com/v1/metrics/native-token-fdv',
  headers={'x-api-key': 'YOUR_API_KEY'},
  params={'project': 'solana'}
)
data = r.json()
```
