## Overview

- **Denomination:** Count
- **Type:** Integer
- **Interval:** Daily
- **Source:** Blockworks

## Example Request

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworks.com/v1/metrics/active-address-total?project=zksync"
```

```typescript TypeScript
const res = await fetch(
  'https://api.blockworks.com/v1/metrics/active-address-total?project=zksync',
  { headers: { 'x-api-key': 'YOUR_API_KEY' } }
)
const data = await res.json()
```

```python Python
import requests

r = requests.get(
  'https://api.blockworks.com/v1/metrics/active-address-total',
  headers={'x-api-key': 'YOUR_API_KEY'},
  params={'project': 'zksync'}
)
data = r.json()
```
