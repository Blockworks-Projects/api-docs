## Overview

- **Denomination:** Count
- **Type:** Integer
- **Interval:** Daily
- **Source:** Blockworks

## Example Request

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworks.com/v1/metrics/etf-num-products?project=harvest-portfolios-group"
```

```typescript TypeScript
const res = await fetch(
  'https://api.blockworks.com/v1/metrics/etf-num-products?project=harvest-portfolios-group',
  { headers: { 'x-api-key': 'YOUR_API_KEY' } }
)
const data = await res.json()
```

```python Python
import requests

r = requests.get(
  'https://api.blockworks.com/v1/metrics/etf-num-products',
  headers={'x-api-key': 'YOUR_API_KEY'},
  params={'project': 'harvest-portfolios-group'}
)
data = r.json()
```
