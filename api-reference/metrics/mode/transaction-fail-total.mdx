## Overview

- **Denomination:** Count
- **Type:** Integer
- **Interval:** Daily
- **Source:** Blockworks

## Example Request

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworks.com/v1/metrics/transaction-fail-total?project=mode"
```

```typescript TypeScript
const res = await fetch(
  'https://api.blockworks.com/v1/metrics/transaction-fail-total?project=mode',
  { headers: { 'x-api-key': 'YOUR_API_KEY' } }
)
const data = await res.json()
```

```python Python
import requests

r = requests.get(
  'https://api.blockworks.com/v1/metrics/transaction-fail-total',
  headers={'x-api-key': 'YOUR_API_KEY'},
  params={'project': 'mode'}
)
data = r.json()
```

</CodeGroup>

## Example Response

```json
{
  "mode": []
}
```

## Notes
- Intervals are daily unless otherwise noted.
- Data is updated daily and may be revised after late-arriving data.
