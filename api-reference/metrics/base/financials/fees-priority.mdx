title: "Priority Fees"
description: "An additional transaction fee to incentivize validators to prioritize and execute transactions."
openapi: 'GET /v1/metrics/fees-priority'
---

## Overview

- **Unit:** native units
- **Interval:** daily
- **Source:** Blockworks
