---
title: 'Fee Average'
description: 'Average of the sum of the base fee and priority fee that day in native chain token'
openapi: 'GET /v1/metrics/fee-average'
---

## Overview

- **Unit:** native units
- **Aggregation:** daily
- **Source:** Blockworks



## Example Response

```json
{
  "base": [
    { "date": "2025-08-01", "value": 592.3947 },
    { "date": "2025-08-02", "value": 605.2811 },
    { "date": "2025-08-03", "value": 618.0204 },
    { "date": "2025-08-04", "value": 598.472 }
  ]
}
```

## Notes
- Daily aggregates unless otherwise noted.
- Use USD conversion with [`fee-average-usd`](/api-reference/metrics/base/fee-average-usd) for fiat value.
- Data is updated daily and may be revised after late-arriving data.
