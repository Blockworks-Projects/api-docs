---
title: "Fee Median"
description: "Median of the sum of the base fee and priority fee that day in native chain token"
openapi: 'GET /v1/metrics/fee-median'
---

## Overview

- **Unit:** native units
- **Interval:** daily
- **Source:** Blockworks

## Example Response

```json
{
  "base": [
    {
      "date": "2025-08-14",
      "value": 0.0000011426033865825474
    },
    {
      "date": "2025-08-15",
      "value": 3.58596750299492e-7
    },
    {
      "date": "2025-08-16",
      "value": 1.658358622221183e-7
    },
    {
      "date": "2025-08-17",
      "value": 1.9498189923153796e-7
    },
    {
      "date": "2025-08-18",
```

## Notes
- daily aggregates unless otherwise noted.
- Data is updated daily and may be revised after late-arriving data.


<Note>For fiat values, see **[Fee Median USD](./fee-median-usd)**.</Note>
