---
title: "Fee Median"
description: "Median of the sum of the base fee and priority fee that day in native chain token"
openapi: 'GET /v1/metrics/fee-median'
---

## Overview

- **Unit:** native units
- **Interval:** daily
- **Source:** Blockworks

## Example Response

```json
{
  "base": [
    {
      "date": "2025-08-13",
      "value": 0.000001194228026156742
    },
    {
      "date": "2025-08-14",
      "value": 0.0000011353442667027885
    },
    {
      "date": "2025-08-15",
      "value": 3.5722627908585397e-7
    },
    {
      "date": "2025-08-16",
      "value": 1.634167437591202e-7
    },
    {
      "date": "2025-08-17",
      "value": 1.2934840061386547e-7
    }
  ]

```

## Notes
- daily aggregates unless otherwise noted.
- Data is updated daily and may be revised after late-arriving data.


<Note>For fiat values, see **[Fee Median USD](./fee-median-usd)**.</Note>
