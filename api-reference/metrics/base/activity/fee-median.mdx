---
title: "Fee Median"
description: "Median of the sum of the base fee and priority fee that day in native chain token"
openapi: 'GET /v1/metrics/fee-median'
---

## Overview

- **Unit:** Native units
- **Aggregation:** Sum
- **Source:** Blockworks

```json
{
  "base": [
    {
      "date": "2025-08-11",
      "value": 9.247979572871473e-7

    {
```

## Notes
- Sum aggregates unless otherwise noted.
- Data is updated daily and may be revised after late-arriving data.


<Note>For fiat values, see **[Fee Median USD](./fee-median-usd)**.</Note>
