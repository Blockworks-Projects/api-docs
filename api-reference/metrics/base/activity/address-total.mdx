openapi: 'GET /v1/metrics/address-total'
---

## Overview

- **Unit:** count
- **Interval:** daily
- **Source:** Blockworks

## Example Request

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworks.com/v1/metrics/address-total?project=base"
```

```typescript TypeScript
const res = await fetch(
  'https://api.blockworks.com/v1/metrics/address-total?project=base',
  { headers: { 'x-api-key': 'YOUR_API_KEY' } }
)
const data = await res.json()
```

```python Python
import requests

r = requests.get(
  'https://api.blockworks.com/v1/metrics/address-total',
  headers={'x-api-key': 'YOUR_API_KEY'},
  params={'project': 'base'}
)
data = r.json()
```
