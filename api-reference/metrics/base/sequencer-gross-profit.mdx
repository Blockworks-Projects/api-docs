## Overview

- **Denomination:** USD
- **Type:** USD Value
- **Interval:** Daily
- **Source:** Blockworks

## Example Request

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworks.com/v1/metrics/sequencer-gross-profit?project=base"
```

```typescript TypeScript
const res = await fetch(
  'https://api.blockworks.com/v1/metrics/sequencer-gross-profit?project=base',
  { headers: { 'x-api-key': 'YOUR_API_KEY' } }
)
const data = await res.json()
```

```python Python
import requests

r = requests.get(
  'https://api.blockworks.com/v1/metrics/sequencer-gross-profit',
  headers={'x-api-key': 'YOUR_API_KEY'},
  params={'project': 'base'}
)
data = r.json()
```
