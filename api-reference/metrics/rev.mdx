## Basic Example

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworksresearch.com/v1/metrics/rev?project=Ethereum"
```

```typescript TypeScript
const response = await fetch(
  'https://api.blockworksresearch.com/v1/metrics/rev?project=Ethereum', {
  headers: { 'x-api-key': 'YOUR_API_KEY' }
})

const data = await response.json()
```

```python Python
import requests

response = requests.get(
    'https://api.blockworksresearch.com/v1/metrics/rev',
    headers={'x-api-key': 'YOUR_API_KEY'},
    params={'project': 'Ethereum'}
)

data = response.json()
```
