## Overview

- **Denomination:** ETH
- **Type:** Decimal
- **Interval:** Daily
- **Source:** Blockworks

## Example Request

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworks.com/v1/metrics/token-supply?project=ethereum"
```

```typescript TypeScript
const res = await fetch(
  'https://api.blockworks.com/v1/metrics/token-supply?project=ethereum',
  { headers: { 'x-api-key': 'YOUR_API_KEY' } }
)
const data = await res.json()
```

```python Python
import requests

r = requests.get(
  'https://api.blockworks.com/v1/metrics/token-supply',
  headers={'x-api-key': 'YOUR_API_KEY'},
  params={'project': 'ethereum'}
)
data = r.json()
```
