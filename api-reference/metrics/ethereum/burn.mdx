---
title: 'Burn'
description: 'The total amount of the native token burned transaction fees.'
openapi: 'GET /v1/metrics/burn'
---

## Overview

- **Unit:** native units
- **Aggregation:** daily
- **Source:** Blockworks



## Example Response

```json
{
  "ethereum": [
    { "date": "2025-08-01", "value": 592.3947 },
    { "date": "2025-08-02", "value": 605.2811 },
    { "date": "2025-08-03", "value": 618.0204 },
    { "date": "2025-08-04", "value": 598.472 }
  ]
}
```

## Notes
- Daily aggregates unless otherwise noted.
- Use USD conversion with [`burn-usd`](/api-reference/metrics/ethereum/burn-usd) for fiat value.
- Data is updated daily and may be revised after late-arriving data.
