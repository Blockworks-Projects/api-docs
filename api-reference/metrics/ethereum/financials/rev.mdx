---
title: "Real Economic Value"
description: "Network REV (Real Economic Value) is a standardized metric that tracks blockchain value accrual generated by user activity. REV consists of both in-protocol transaction fees and out-of-protocol tips that users pay for transaction execution, so it measures the monetary demand to transact onchain."
openapi: 'GET /v1/metrics/rev'
---

## Overview

- **Unit:** Native units
- **Aggregation:** Sum
- **Source:** Blockworks
