---
title: 'REV'
description: 'Network REV (Real Economic Value) is a standardized metric that tracks blockchain value accrual generated by user activity. REV consists of both in-protocol transaction fees and out-of-protocol tips that users pay for transaction execution, so it measures the monetary demand to transact onchain.'
openapi: 'GET /v1/metrics/rev'
---

## Overview

- **Unit:** native units
- **Aggregation:** daily
- **Source:** Blockworks



## Example Response

```json
{
  "ethereum": [
    { "date": "2025-08-01", "value": 592.3947 },
    { "date": "2025-08-02", "value": 605.2811 },
    { "date": "2025-08-03", "value": 618.0204 },
    { "date": "2025-08-04", "value": 598.472 }
  ]
}
```

## Notes
- Daily aggregates unless otherwise noted.
- Use USD conversion with [`rev-usd`](/api-reference/metrics/ethereum/rev-usd) for fiat value.
- Data is updated daily and may be revised after late-arriving data.
