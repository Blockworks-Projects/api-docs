## Overview

- **Unit:** USD
- **Aggregation:** daily
- **Source:** Blockworks

## Notes
- Values are reported in **USD** where applicable.
- Native counterpart: [`fees-priority`](/api-reference/metrics/ethereum/fees-priority)
