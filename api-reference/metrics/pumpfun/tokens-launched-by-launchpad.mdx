## Overview

- **Denomination:** Count
- **Type:** Integer
- **Interval:** Daily
- **Source:** Blockworks

## Example Request

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworks.com/v1/metrics/tokens-launched-by-launchpad?project=pumpfun"
```

```typescript TypeScript
const res = await fetch(
  'https://api.blockworks.com/v1/metrics/tokens-launched-by-launchpad?project=pumpfun',
  { headers: { 'x-api-key': 'YOUR_API_KEY' } }
)
const data = await res.json()
```

```python Python
import requests

r = requests.get(
  'https://api.blockworks.com/v1/metrics/tokens-launched-by-launchpad',
  headers={'x-api-key': 'YOUR_API_KEY'},
  params={'project': 'pumpfun'}
)
data = r.json()
```

</CodeGroup>
