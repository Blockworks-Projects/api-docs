## Overview

Fetch a single asset by **ID** or **slug**. You can optionally expand related objects (e.g. supply) to avoid making multiple requests. Price and market cap are expanded by default.

## Quick summary

| What | Details |
| - | - |
| Path | `/v1/assets/{idOrSlug}` |
| Auth | `x-api-key` header |
| Expandable fields | `addresses`, `chains`, `addresses.chain`, `is_favorite`, `markets`, `market_cap`, `ohlcv_last_24_h`, `price`, `sector`, `supply` |

## Basic Example

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \\
  "https://api.blockworks.com/v1/assets/eth?expand=supply"
```

```typescript TypeScript
const res = await fetch(
  'https://api.blockworks.com/v1/assets/eth?expand=supply',
  { headers: { 'x-api-key': 'YOUR_API_KEY' } }
)
const data = await res.json()
```

```python Python
import requests

r = requests.get(
  'https://api.blockworks.com/v1/assets/eth',
  headers={'x-api-key': 'YOUR_API_KEY'},
  params={'expand': 'supply'}
)
data = r.json()
```

</CodeGroup>
