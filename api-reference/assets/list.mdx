## Basic Example

<CodeGroup>

```typescript TypeScript
const response = await fetch(
  'https://api.blockworksresearch.com/v3/assets?limit=20', {
  headers: { 'x-api-key': 'YOUR_API_KEY' }
})

const data = await response.json()
```

```python Python
import requests

response = requests.get(
    'https://api.blockworksresearch.com/v3/assets',
    headers = { 'x-api-key': 'YOUR_API_KEY' },
    params = { 'limit': 20 }
)

data = response.json()
```

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  "https://api.blockworksresearch.com/v3/assets?limit=20"
```

</CodeGroup>

## Configuration

### Query Parameters

| Parameter | Type | Description | Default |
|-----------|------|-------------|---------|
| `page` | integer | Page number (1-indexed) | 1 |
| `limit` | integer | Items per page (max 100) | 10 |
| `code` | string | Filter by asset code (e.g., "BTC") | - |
| `slug` | string | Filter by asset slug | - |
| `category` | string | Filter by asset category | - |
| `favorites` | boolean | Show only favorite assets | false |
| `expand` | array | Expand related data | - |

### Expansion Options

Use the `expand` parameter to include additional related data for each asset:

| Value | Description |
|-------|-------------|
| `price` | Current price information |
| `market_cap` | Market capitalization data |
| `sector` | Sector classification |
| `markets` | Trading market data |
| `supply` | Supply information |
| `is_favorite` | Favorite status |

## Examples

### Filter by Category

<CodeGroup>

```typescript TypeScript
const response = await fetch('https://api.blockworksresearch.com/v3/assets?category=Infrastructure&limit=50', {
  headers: { 'x-api-key': 'YOUR_API_KEY' }
})

const data = await response.json()
```

```python Python
response = requests.get(
    'https://api.blockworksresearch.com/v3/assets',
    headers={'x-api-key': 'YOUR_API_KEY'},
    params={'category': 'Infrastructure', 'limit': 50}
)
```

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.blockworksresearch.com/v3/assets?category=Infrastructure&limit=50"
```

</CodeGroup>
