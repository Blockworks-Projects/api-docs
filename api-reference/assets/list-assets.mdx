## GET /v3/assets

Retrieve a paginated list of crypto assets with optional filtering and expansion of related data.

### Request

```bash
GET https://rest.staging.blockworksresearch.com/v3/assets
```

### Authentication

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://rest.staging.blockworksresearch.com/v3/assets"
```

```typescript TypeScript
const response = await fetch('https://rest.staging.blockworksresearch.com/v3/assets', {
  headers: {
    'x-api-key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});
```

```python Python
import requests

headers = {
    'x-api-key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
}

response = requests.get(
    'https://rest.staging.blockworksresearch.com/v3/assets',
    headers=headers
)
```

</CodeGroup>

### Query Parameters

| Parameter | Type | Description | Default |
|-----------|------|-------------|---------|
| `page` | integer | Page number (1-indexed) | 1 |
| `limit` | integer | Items per page (max 100) | 10 |
| `code` | string | Filter by asset code/symbol | - |
| `slug` | string | Filter by asset slug | - |
| `is_supported` | boolean | Filter by supported status | - |
| `sector` | string | Filter by sector | - |
| `expand` | array | Expand related data | - |
| `sort` | string | Sort by field | - |

### Expansion Options

Use the `expand` parameter to include additional data:

| Value | Description |
|-------|-------------|
| `price` | Current price information |
| `market_cap` | Market capitalization data |
| `sector` | Sector classification |
| `addresses` | Contract addresses |
| `markets` | Trading market information |

### Examples

#### Basic Request

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://rest.staging.blockworksresearch.com/v3/assets?limit=20"
```

```typescript TypeScript
const response = await fetch('https://rest.staging.blockworksresearch.com/v3/assets?limit=20', {
  headers: {
    'x-api-key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
console.log(`Found ${data.data.length} assets`);
```

```python Python
import requests

headers = {'x-api-key': 'YOUR_API_KEY'}
params = {'limit': 20}

response = requests.get(
    'https://rest.staging.blockworksresearch.com/v3/assets',
    headers=headers,
    params=params
)

data = response.json()
print(f"Found {len(data['data'])} assets")
```

</CodeGroup>

#### With Expanded Data

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://rest.staging.blockworksresearch.com/v3/assets?expand=price,market_cap,sector&limit=10"
```

```typescript TypeScript
const response = await fetch('https://rest.staging.blockworksresearch.com/v3/assets?expand=price,market_cap,sector&limit=10', {
  headers: {
    'x-api-key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
data.data.forEach(asset => {
  console.log(`${asset.name}: $${asset.price} (${asset.sector})`);
});
```

```python Python
headers = {'x-api-key': 'YOUR_API_KEY'}
params = {
    'expand': 'price,market_cap,sector',
    'limit': 10
}

response = requests.get(
    'https://rest.staging.blockworksresearch.com/v3/assets',
    headers=headers,
    params=params
)

data = response.json()
for asset in data['data']:
    print(f"{asset['name']}: ${asset['price']} ({asset['sector']})")
```

</CodeGroup>

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique asset identifier |
| `slug` | string | URL-friendly asset identifier |
| `name` | string | Full asset name |
| `symbol` | string | Trading symbol |
| `code` | string | Asset code |
| `is_supported` | boolean | Whether asset is supported |
| `price` | number | Current price (if expanded) |
| `market_cap` | number | Market capitalization (if expanded) |
| `sector` | string | Asset sector (if expanded) |
| `addresses` | object | Contract addresses (if expanded) |
| `markets` | array | Trading markets (if expanded) |

### Common Use Cases

#### Portfolio Tracking

Get current prices for your portfolio assets:

```typescript
const portfolioSlugs = ['bitcoin', 'ethereum', 'solana'];
const slugFilter = portfolioSlugs.join(',');

const response = await fetch(`https://rest.staging.blockworksresearch.com/v3/assets?slug=${slugFilter}&expand=price,market_cap`, {
  headers: { 'x-api-key': 'YOUR_API_KEY' }
});
```

#### Sector Analysis

Analyze assets by sector:

```python
sectors = ['DeFi', 'Layer 1', 'Layer 2']
sector_data = {}

for sector in sectors:
    response = requests.get(
        'https://rest.staging.blockworksresearch.com/v3/assets',
        headers={'x-api-key': 'YOUR_API_KEY'},
        params={'sector': sector, 'expand': 'price,market_cap', 'limit': 50}
    )
    sector_data[sector] = response.json()['data']
```

#### Market Screening

Screen for assets meeting specific criteria:

```typescript
// Find DeFi assets with market cap > $1B
const response = await fetch('https://rest.staging.blockworksresearch.com/v3/assets?sector=DeFi&expand=market_cap&limit=100', {
  headers: { 'x-api-key': 'YOUR_API_KEY' }
});

const data = await response.json();
const largeCap = data.data.filter(asset => asset.market_cap > 1000000000);
```
