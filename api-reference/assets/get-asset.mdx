## GET /v3/assets/{idOrSlug}

Retrieve detailed information about a specific crypto asset by its ID or slug.

### Request

```bash
GET https://rest.staging.blockworksresearch.com/v3/assets/{idOrSlug}
```

### Path Parameters

| Parameter | Type | Description | Required |
|-----------|------|-------------|----------|
| `idOrSlug` | string | Asset ID or slug identifier | Yes |

### Authentication

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://rest.staging.blockworksresearch.com/v3/assets/bitcoin"
```

```typescript TypeScript
const response = await fetch('https://rest.staging.blockworksresearch.com/v3/assets/bitcoin', {
  headers: {
    'x-api-key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});
```

```python Python
import requests

headers = {
    'x-api-key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
}

response = requests.get(
    'https://rest.staging.blockworksresearch.com/v3/assets/bitcoin',
    headers=headers
)
```

</CodeGroup>

### Query Parameters

| Parameter | Type | Description | Default |
|-----------|------|-------------|---------|
| `expand` | array | Expand related data | - |

### Expansion Options

Use the `expand` parameter to include additional data:

| Value | Description |
|-------|-------------|
| `price` | Current price information |
| `market_cap` | Market capitalization data |
| `sector` | Sector classification |
| `addresses` | Contract addresses across chains |
| `markets` | Trading market information |
| `price_history` | Historical price data |
| `metrics` | Key asset metrics |

#### With Full Expansion

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://rest.staging.blockworksresearch.com/v3/assets/ethereum?expand=price,market_cap,sector,addresses,markets"
```

```typescript TypeScript
const response = await fetch('https://rest.staging.blockworksresearch.com/v3/assets/ethereum?expand=price,market_cap,sector,addresses,markets', {
```

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique asset identifier |
| `slug` | string | URL-friendly asset identifier |
| `name` | string | Full asset name |
| `symbol` | string | Trading symbol |
| `code` | string | Asset code |
| `is_supported` | boolean | Whether asset is supported |
| `description` | string | Asset description |
| `price` | number | Current price (if expanded) |
| `market_cap` | number | Market capitalization (if expanded) |
| `circulating_supply` | number | Circulating supply |
| `total_supply` | number | Total supply |
| `max_supply` | number | Maximum supply |
| `sector` | string | Asset sector (if expanded) |
| `tags` | array | Asset tags |
| `addresses` | object | Contract addresses (if expanded) |
| `markets` | array | Trading markets (if expanded) |
| `price_history` | object | Historical prices (if expanded) |
| `metrics` | object | Key metrics (if expanded) |
| `last_updated` | string | Last update timestamp |

### Common Use Cases

#### Asset Overview Dashboard

Get comprehensive asset information for a dashboard:

```typescript
async function getAssetOverview(slug: string) {
  const response = await fetch(`https://rest.staging.blockworksresearch.com/v3/assets/${slug}?expand=price,market_cap,sector,metrics`, {
    headers: { 'x-api-key': 'YOUR_API_KEY' }
  });
  
  const data = await response.json();
  const asset = data.data;
  
  return {
    name: asset.name,
    symbol: asset.symbol,
    price: asset.price,
    marketCap: asset.market_cap,
    sector: asset.sector,
    priceChange24h: asset.metrics.price_change_24h,
    volume24h: asset.metrics.volume_24h,
    rank: asset.metrics.market_cap_rank
  };
}

// Usage
const bitcoinOverview = await getAssetOverview('bitcoin');
```

#### Portfolio Asset Details

Get detailed information for portfolio assets:

```python
def get_portfolio_asset_details(assets: list):
    """Get detailed information for portfolio assets"""
    portfolio_details = []
    
    for asset_slug in assets:
        response = requests.get(
            f'https://rest.staging.blockworksresearch.com/v3/assets/{asset_slug}',
            headers={'x-api-key': 'YOUR_API_KEY'},
            params={'expand': 'price,market_cap,metrics,addresses'}
        )
        
        if response.status_code == 200:
            asset_data = response.json()['data']
            portfolio_details.append({
                'name': asset_data['name'],
                'symbol': asset_data['symbol'],
                'price': asset_data['price'],
                'market_cap': asset_data['market_cap'],
                'volatility': asset_data['metrics']['volatility_30d'],
                'addresses': asset_data['addresses']
            })
    
    return portfolio_details

# Usage
portfolio_assets = ['bitcoin', 'ethereum', 'solana']
details = get_portfolio_asset_details(portfolio_assets)
```

#### Market Data Analysis

Analyze trading markets for an asset:

```python
def analyze_asset_markets(slug: str):
    """Analyze trading markets for an asset"""
    response = requests.get(
        f'https://rest.staging.blockworksresearch.com/v3/assets/{slug}',
        headers={'x-api-key': 'YOUR_API_KEY'},
        params={'expand': 'markets,price,metrics'}
    )
    
    if response.status_code == 200:
        asset_data = response.json()['data']
        
        # Sort markets by volume
        markets = sorted(asset_data['markets'], 
                        key=lambda x: x['volume_24h'], 
                        reverse=True)
        
        print(f"Top markets for {asset_data['name']}:")
        for market in markets[:5]:
            print(f"  {market['exchange']}: {market['pair']} - ${market['volume_24h']:,.0f}")
        
        return markets
    
    return None

# Usage
btc_markets = analyze_asset_markets('bitcoin')
```
